# GrihaLedger

GrihaLedger is a multi-tenant household finance system designed with real-world data engineering principles.

## Phase 1 (March 2026)
Core transactional system:
- Households
- Members
- Accounts
- Transactions
- Visibility rules
- Balance tracking

Future phases:
- ELT with dbt
- Orchestration with Airflow
- Data quality enforcement
- Scale testing & partitioning
